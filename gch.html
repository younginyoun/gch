단계적으로 탐색하는 계층형 전자제품 쇼핑몰

본 페이지는 웹사이트 정보 구조 비교 실험을 위한 계층형 프로토타입이다. 홈에서 카테고리를 선택하고, 카테고리 목록에서 상품을 고른 뒤, 상세 페이지에서 구매 정보를 확인하는 구조로 설계했다.

홈 → 카테고리 → 목록 → 상세 전체 상품 28개 카테고리 7개 실험용 단계 탐색 구조

  • 4~6번: 복합 조건 비교 문제
  • 문제 번호 입력 시 상단에 자동 표시
